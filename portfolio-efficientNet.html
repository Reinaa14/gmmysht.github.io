Project Information

  • Category: Deep Learning
  • Client: Final Assigment Research
  • Project date: October, 2023 - January, 2024

Project Overview

This project focused on building a deep learning classification model to detect brain tumors using the EfficientNet-B7 architecture. The dataset combined X-ray brain images from RSU Dr. Slamet Garut and publicly available images from Kaggle to ensure diverse and representative data.

Project Objectives

  • Develop a robust classification model to detect brain tumors in X-ray images.
  • Analyze the performance impact of different optimizers on model accuracy.
  • Provide insights for potential implementation in medical diagnosis support systems.

Scope of Work

  • Dataset preparation from RSU Dr. Slamet Garut and Kaggle.
  • Model development using EfficientNet-B7 architecture.
  • Hyperparameter tuning for optimizer, epoch, learning rate, and batch size.
  • Performance evaluation and analysis of training results.

Key Findings

Best Accuracy

95.29% accuracy using Adam optimizer, 15 epochs, learning rate 0.001, and batch size 16.

Lowest Accuracy

89.41% accuracy using Nadam optimizer, 20 epochs, learning rate 0.001, and batch size 16.

Optimizer Performance

Adam outperformed Nadam and Adamax due to adaptive learning rates and faster convergence.

Potential Application

EfficientNet-B7 shows promise for brain tumor detection, with further improvements possible through optimized parameters and datasets.

Technology Stack

  • Framework: TensorFlow / Keras
  • Model: EfficientNet-B7
  • Programming Language: Python
  • Tools: Google Colab, Matplotlib for result visualization

Your Role in the Project

  • Prepared and preprocessed datasets for training and testing.
  • Developed and trained the EfficientNet-B7 model with various hyperparameters.
  • Analyzed training results to determine optimal model configuration.
  • Documented findings and suggested improvements for future development.
