Project Overview

This project developed a diabetic retinopathy detection model using Mask R-CNN for feature extraction and XGBoost for classification. The dataset consisted of 5,002 retinal fundus images (.png) divided into two classes: Diabetic (2,501 images) and Normal (2,501 images), sourced from Kaggle and organized into class folders. The dataset was split into 80% training and 20% testing sets.

Project Objectives

  • Build a robust detection model for diabetic retinopathy using deep learning and ensemble classification.
  • Enhance dataset variation through preprocessing and data augmentation to improve model generalization.
  • Evaluate model performance and determine the best hyperparameter configuration.

Scope of Work

  • Dataset preparation and class folder organization.
  • Image preprocessing and augmentation for balanced data.
  • Feature extraction using Mask R-CNN architecture with TensorFlow v1.
  • Classification with XGBoost including hyperparameter tuning.
  • Model evaluation using accuracy, precision, recall, and F1-score metrics.

Key Findings

Best Model Performance

Validation accuracy: 98.62%, test accuracy: 99.07% using n_estimators=400, max_depth=5, learning_rate=0.05, subsample=0.4.

Hyperparameter Impact

Increasing n_estimators improved performance up to a point before risking overfitting and increased computational cost.

Feature Extraction with Mask R-CNN

Utilized for extracting relevant retinal features prior to classification, leveraging TensorFlow v1 implementation.

Evaluation Metrics

Model performance assessed using accuracy, precision, recall, and F1-score for robust validation.

Technology Stack
