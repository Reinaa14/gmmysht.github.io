Project Overview

This project focuses on building a classification model for brain tumor detection using the DenseNet-201 architecture. The dataset consisted of 2,781 brain tumor images (.jpg) divided into three classes: Meningioma, Glioma, and Pituitary. Images were sourced from Kaggle and validated by radiologists at RSUD Dr. Slamet, Garut. An additional 32 images from the same hospital were used for testing.

Project Objectives

  • Develop an accurate model to classify brain tumor images into three categories.
  • Balance dataset classes through data augmentation to prevent overfitting.
  • Analyze hyperparameter effects on model performance for optimal configuration.

Scope of Work

  • Data preprocessing and augmentation to balance class distribution.
  • Dataset splitting: 80% for training, 20% for testing.
  • Model training using DenseNet-201 with various hyperparameter configurations.
  • Performance evaluation to determine the best model setup.

Key Findings

Best Model

Adam optimizer, 20 epochs, learning rate 0.5, batch size 16 with validation accuracy of 81.29%.

Lowest Performance

Adam optimizer, 50 epochs, learning rate 0.5, batch size 16 with validation accuracy of 74.28%.

Optimizer Analysis

Adam performed better than Nadam and Adamax due to its adaptive learning rate and fast convergence, suitable for DenseNet-201's complexity.

Hyperparameter Tuning

Hyperparameter adjustment was critical for optimizing performance in complex models like DenseNet-201.

Technology Stack

  • Model: DenseNet-201
  • Programming Language: Python
  • Tools: Google Colab, Matplotlib for result visualization
